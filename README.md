#  Enhancing Industrial Wireless Sensor Network Security Using Machine Learning and Deep Learning Models

##  Overview

Industrial Wireless Sensor Networks (IWSNs) play a vital role in Industry 4.0 by enabling real-time monitoring, automation, and communication among industrial devices. However, these networks are highly vulnerable to cyberattacks such as Denial of Service (DoS), ransomware, insider attacks, and Advanced Persistent Threats (APT).

This project develops an intelligent Intrusion Detection System (IDS) using Machine Learning and Deep Learning techniques to detect malicious activities within industrial wireless sensor networks. The system analyzes network traffic, classifies attacks, and compares the performance of multiple machine learning algorithms to improve cybersecurity in industrial environments.
